    「数当てゲームの説明」

  • 乱数で 1〜10の値を1つ決めて、「答え」とする
  • ユーザは答えを予想して、数を最大で3回入力できる
  • ユーザの入力が外れた場合は，入力よりも答えが大きいか小さいか表示する
  • 3回以内に答えの数を当てればユーザの勝ち、そうでなければ負け
  • ページには，テキストボックスとボタンが設置されている
    • ユーザはテキストボックスに予想した数を入力
    • ボタンを押すと判定

    第2回 CSS による Web ページの装飾

    今回はCSSについて学びました。CSS（Cascading Style Sheets）とは、 Webサイトのサイズや色、レイアウトなどを設定 するためのプログラミング言語であり、CSSは「シーエスエス」や「スタイルシート」などと呼ばれており、背景の色の変更や画像の設置、文字のフォントや色などの幅広いデザインを定義する際に使用されている。CSSを利用してページを見やすく作成した。
