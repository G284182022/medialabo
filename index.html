「Webコンテンツ作成」の成果

本テーマの学習目的

本テーマでは、HTML,CSS,JavaScriptの基本を学び、これらを組み合わせてWebページを構成する方法を学びます。

学習目標:

  • HTML によって Web ページの構造を記述する基本的な方法がわかる
  • CSS によって Web ページの見た目を設定する基本的な方法がわかる
  • JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる
  • HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できる

プロジェクト課題について

WWWのサーバへ検索し、サーバから届いたデータを Webページとして整形するような Webアプリケーションを作成する。自分は世界の天気についてWebページの作成せました。全6回の講義を通じてプロジェクトを作成しました。第1回ではデータを選び、Webアプリケーションのページテンプレートの作成をし、第2回ではCSSについて学び、Webページの装飾をしました。テキストや背景をカラーしたり、フォントを変えたりしてデザインしました。第3回では、JavaScriptを学び、JSを使ってデータ項目全てをコンソールで出力できるようにプログラムを改良しました。第4回では、DOMとJavaScriptの勉強をし、プログラムに修正をし、div要素を入れ、コンソールに表示していたデータ項目をdiv要素に表示できるようにした。第5回はイベント処理について学び、inputやbotton要素を使い、検索欄を作りました。そして第6回でこのプロジェクト課題を完成させた。

数当てゲーム

数当てゲームは1〜10で設定された数字を3回の予想で数字を当てるゲームです。3回以内に出れば、ユーザが勝ちで、そうでなければ負けになります。また、予想より大きいか小さいかも表示する。最初はコンソールに出力されるようにプログラムを作り、それをコンソールから表示されるようにプログラムを修正させ、プログラムを短縮化した。最後にボタンを作り、ボタンを押すと検索結果が出るようにプログラムを作成した。

「数当てゲームの説明」

  • 乱数で 1〜10の値を1つ決めて、「答え」とする
  • ユーザは答えを予想して、数を最大で3回入力できる
  • ユーザの入力が外れた場合は，入力よりも答えが大きいか小さいか表示する
  • 3回以内に答えの数を当てればユーザの勝ち、そうでなければ負け
  • ページには，テキストボックスとボタンが設置されている
  • ユーザはテキストボックスに予想した数を入力
  • ボタンを押すと判定
  • 数当てゲーム
